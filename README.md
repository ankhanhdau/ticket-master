# **Ticket Master – A Dockerized Kanban Board for Agile Teams**

## **Tech Stack**
- **Frontend:** React, TypeScript, Tailwind CSS  
- **Backend:** Node.js, Express  
- **Database:** PostgreSQL  
- **Infrastructure:** Docker & Docker Compose  

---

## **Overview**
TicketMaster is a lightweight, full-stack Kanban board designed for agile teams. It supports complete ticket lifecycle management with an intuitive UI and a simple, containerized setup.

---

## **Features**
- **Full CRUD for Tickets** – Create, read, update, and delete tickets.  
- **Kanban Board** – Visualize ticket progress click-to-move interaction.  
- **Optimistic UI Updates** – Instant feedback without waiting for server responses.  
- **Dockerized System** – API and PostgreSQL run inside containers for a consistent environment.

---

## **Getting Started**

### **Clone the Repository**
```bash
git clone https://github.com/ankhanhdau/ticket-master.git
cd ticket-master
docker-compose up --build
```
